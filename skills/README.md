# Skills

Claude Code skills collection.

## research-ideation

Systematic iterative research idea generation with multi-role expert review.

**Install:**
```bash
claude plugins install research-ideation.skill
```

Or install from GitHub:
```bash
claude plugins install https://github.com/wushangruide/skills/raw/main/research-ideation.skill
```

**What it does:**
- Iteratively generates and evaluates research ideas
- Four-role review panel (Domain Expert, Cross-Domain Generalist, Devil's Advocate, Mock Reviewer)
- Weighted scoring with configurable thresholds
- Literature search for novelty checking
- Structured output (idea .md + log.json + SUMMARY.md)

**When it triggers:**
- "generate research ideas systematically"
- "improve my paper's innovation"
- "need structured brainstorming for research directions"
